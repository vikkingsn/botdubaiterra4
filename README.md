# Telegram Bot для управления рассылками

Telegram-бот на Python (aiogram 3.x) для управления персонализированными рассылками с комплексной системой отчетности.

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)
[![aiogram](https://img.shields.io/badge/aiogram-3.10.0-green.svg)](https://docs.aiogram.dev/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Возможности

### Для администратора
- ✅ Создание шаблонов сообщений для рассылок
- ✅ Настройка получателей сводных отчетов
- ✅ Полный доступ ко всем функциям

### Для пользователей
- ✅ Запуск рассылок по своим спискам получателей
- ✅ Получение персональных отчетов
- ✅ Просмотр истории рассылок
- ✅ Управление только своими рассылками

### Основные функции
- 🔍 Автоматическая проверка дублей (предотвращение повторных отправок)
- 📊 Детальная статистика по каждой рассылке
- 📈 Сводные отчеты для администраторов
- ⚠️ Детальная обработка ошибок отправки
- ⏱️ Автоматические задержки между отправками (5-11 минут)

## Установка и запуск

📖 **Подробная инструкция по развертыванию**: см. [DEPLOYMENT.md](DEPLOYMENT.md)

### Быстрый старт:

1. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

2. **Создайте файл `.env`:**
```env
BOT_TOKEN=ваш_токен_от_BotFather
MAIN_ADMIN_ID=ваш_telegram_id
DATABASE_URL=sqlite+aiosqlite:///bot.db
```

3. **Получите токен бота:**
   - Найдите [@BotFather](https://t.me/BotFather) в Telegram
   - Отправьте `/newbot` и следуйте инструкциям

4. **Узнайте свой Telegram ID:**
   - Найдите [@userinfobot](https://t.me/userinfobot)
   - Отправьте `/start` и скопируйте ID

5. **Запустите бота:**
```bash
python3 main.py
```

### Требования:
- Python 3.10+
- Интернет-соединение
- ~100 МБ свободного места

## Использование

### Команды для пользователей

- `/start` - регистрация и приветствие
- `/new_mailing` или кнопка "📧 Новая рассылка" - запуск новой рассылки
- `/my_mailings` или кнопка "📊 Мои рассылки" - история моих рассылок
- `/report <ID>` - просмотр конкретного отчета (например, `/report 123`)
- `/help` - справка

### Команды для администратора

- `/add_template` или кнопка "📝 Шаблоны" - создание шаблона сообщения
- `/set_report_receivers` - настройка получателей сводных отчетов
- `/templates_list` - список всех шаблонов

### Процесс создания рассылки

1. Выберите шаблон из списка
2. Введите список получателей (поддерживаются @username, user_id, ссылки)
3. Подтвердите запуск рассылки
4. Бот автоматически:
   - Проверит дубли
   - Отправит сообщения новым получателям
   - Отправит отчет после завершения

### Формат списка получателей

Поддерживаются следующие форматы:
- `@username`
- `123456789` (user_id)
- `https://t.me/username`
- `t.me/username`

Разделители: запятая, пробел, новая строка

Пример:
```
@user1, 123456789, @user2
https://t.me/user3
```

## База данных

Бот использует SQLite (по умолчанию) или PostgreSQL. Все таблицы создаются автоматически при первом запуске.

### Таблицы:
- `users` - пользователи бота
- `templates` - шаблоны сообщений
- `mailing_campaigns` - информация о рассылках
- `recipients` - получатели с историей
- `report_receivers` - получатели сводных отчетов
- `sending_history` - детальная история отправок

## Логирование

Все действия бота логируются в файл `bot.log` с уровнем детализации, указанным в `config.py`.
