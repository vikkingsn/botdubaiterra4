# Чеклист для запуска бота

Используйте этот чеклист для проверки готовности к запуску:

## ✅ Подготовка

- [ ] Python 3.10+ установлен (`python3 --version`)
- [ ] Проект скопирован/клонирован
- [ ] Находитесь в директории проекта

## ✅ Установка зависимостей

- [ ] Выполнена команда: `pip install -r requirements.txt`
- [ ] Все пакеты установлены без ошибок
- [ ] (Опционально) Создано виртуальное окружение

## ✅ Настройка бота

- [ ] Создан бот через @BotFather
- [ ] Получен токен бота (формат: `123456789:ABC...`)
- [ ] Узнан ваш Telegram ID (через @userinfobot)
- [ ] Создан файл `.env` в корне проекта
- [ ] В `.env` указан `BOT_TOKEN`
- [ ] В `.env` указан `MAIN_ADMIN_ID`
- [ ] В `.env` указан `DATABASE_URL`

## ✅ Проверка перед запуском

- [ ] Файл `.env` существует и заполнен
- [ ] Токен бота правильный (начинается с цифр)
- [ ] Telegram ID правильный (только цифры)
- [ ] Нет синтаксических ошибок в коде

## ✅ Запуск

- [ ] Выполнена команда: `python3 main.py`
- [ ] В логах нет критических ошибок
- [ ] Бот отвечает на `/start` в Telegram
- [ ] База данных создалась (`bot.db` существует)

## ✅ Первичная настройка (для администратора)

- [ ] Создан хотя бы один шаблон (`/add_template`)
- [ ] (Опционально) Настроены получатели отчетов (`/set_report_receivers`)

## ✅ Тестирование

- [ ] Создана тестовая рассылка
- [ ] Получен отчет о рассылке
- [ ] Проверена работа проверки дублей

---

## Если что-то не работает:

1. **Проверьте логи**: `tail -f bot.log`
2. **Проверьте `.env` файл**: убедитесь что токен и ID правильные
3. **Проверьте интернет-соединение**
4. **Перезапустите бота**: остановите (Ctrl+C) и запустите снова
5. **Проверьте версию Python**: должна быть 3.10+

---

## Готово! 🎉

Если все пункты отмечены - бот готов к работе!
