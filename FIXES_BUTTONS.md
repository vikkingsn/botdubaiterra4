## ✅ Проверка работы:

1. **Настройки** (только для админа):
   - Нажмите "⚙️ Настройки"
   - Должен показаться список получателей отчетов и команды

2. **Помощь**:
   - Нажмите "ℹ️ Помощь"
   - Должна показаться подробная справка с примерами

3. **Мои рассылки**:
   - Нажмите "📊 Мои рассылки"
   - Должен показаться список рассылок или сообщение, если их нет

4. **Новая рассылка**:
   - Нажмите "📧 Новая рассылка"
   - Выберите шаблон
   - Должно появиться сообщение с инструкциями по вводу получателей
   - Введите получателей
   - Должно появиться превью рассылки
