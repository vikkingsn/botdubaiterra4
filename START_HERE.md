## 🔧 Быстрая проверка готовности:

Выполните эти команды в терминале:

```bash
# 1. Проверка Python
python3 --version
# Должно показать: Python 3.10.x или выше

# 2. Переход в директорию проекта
cd /Users/aspbr/Documents/курсор

# 3. Установка зависимостей
pip install -r requirements.txt

# 4. Проверка наличия .env
ls -la .env
# Должен существовать файл .env

# 5. Запуск
python3 main.py
```

---

## ❓ Проблемы?

### "python3: command not found"
→ Установите Python 3.10+ с [python.org](https://www.python.org/downloads/)

### "ModuleNotFoundError"
→ Выполните: `pip install -r requirements.txt`

### "BOT_TOKEN не установлен"
→ Проверьте файл `.env`, убедитесь что он в корне проекта

### Бот не отвечает
→ Проверьте токен в `.env`, перезапустите бота
